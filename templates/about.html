About the Breast Cancer Prediction Tool

Project Information

This application uses machine learning to predict whether a breast mass is benign or malignant based on various features derived from a digitized image of a fine needle aspirate (FNA) of the breast mass.

The Dataset

The model was built using the Breast Cancer Wisconsin (Diagnostic) Data Set available on the UCI Machine Learning Repository. This dataset includes features computed from digitized FNA images of breast mass samples, capturing key characteristics of the cell nuclei. The data provides a foundation for making informed predictions about the malignancy of a breast mass.

The Model

For prediction purposes, we used a Random Forest classifier. This model was trained on 80% of the data and evaluated on the remaining 20%. The classifier achieved a high accuracy rate, with over 95% correct predictions on the test set. This performance demonstrates the model's effectiveness in diagnosing breast cancer from image-based data.

Features Used

The features used in this model include measurements such as:

  • Radius (mean of distances from center to points on the perimeter)
  • Texture (standard deviation of gray-scale values)
  • Perimeter
  • Area
  • Smoothness (local variation in radius lengths)

  • Concavity (severity of concave portions of the contour)
  • Concave points (number of concave portions of the contour)
  • Symmetry
  • Fractal dimension ("coastline approximation" - 1)
